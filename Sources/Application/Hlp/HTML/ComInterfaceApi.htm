COM Interface API Listing

BUSMASTER COM Interface API Listing

  • Connect : To connect BUSMASTER to the bus 
  • ResetHW : To reset the hardware 
  • GetErrorCounter : Getter for both Rx and Tx error, given the channel number
  • SendCANMSg: To send a CAN message 
  • GetMsgInfo : Retrieves the message parameters from the loaded database, given the message name 
  • GetNetworkStatistics :  To get all the statistics of a channel  
  • LoadAllDll :  To load all user DLLs 
  • UnLoadAllDll : To Unload all DLLs 
  • SendKeyValue : To simulate a particular key stroke
  • EnableDisableHandlers : To enable or disable DLL handlers
  • ImportDatabase : To import a database file in BUSMASTER
  • LoadConfiguration : To load a configuration file in BUSMASTER
  • SaveConfiguration : To save current configuration file
  • SaveConfigurationAs : To save current configuration in a particular file
  • GetTxBlockCount : Getter for current transmission block count
  • ClearTxBlockList : Deletes all the transmission blocks
  • StartTxMsgBlock : To start sending messages
  • StopTxMsgBlock : To stop sending messages
  • EnableFilterSch : Enables / disables a particular filtering scheme
  • GetLoggingBlockCount : Getter for total number of logging blocks
  • RemoveLoggingBlock : To remove a particular logging block
  • ClearLoggingBlock : Clears the current logging blocks defined
  • StartLogging : To start logging
  • StopLogging : To stop logging
  • WriteToLogFile : Writes a text in the log file currently open                                                          

SendCANMSg

This function will put the message on the CAN bus. The message structure CAN_MSGS will be filled with ID, length, frame format and data.
Inputs

CAN_MSGS - CAN Message Structure

Cause of failure

Tool not connected

GetMsgInfo

This function will get the details about a database message.
Inputs Message name,                                                                                             Structure sMESSAGESTRUCT { double m_dMessageCode;                                 double m_dNumberOfSignals; double m_dMessageLength;                                   BOOL m_bMessageFrameFormat; int m_nMsgDataFormat; }

Cause of failure

message name not in database,data bese file is not imported to BUSMASTER

 

SendKeyValue

This function can be used to capture any key pressed in client application and pass it to BUSMASTER.The function takes ascii character of the key.Only alphanumeric key is allowed.
Inputs Ascii value of key.

Cause of failure

The Ascii value doesn't corresspond to any Alphanumeric character.


 

LoadConfiguration

This function will load the configuration file specified by client.
Inputs Configuration file path.

Cause of failure

File absent,tool is connected,DLLs are loaded

ImportDatabase

This function will import the specfied file to BUSMASTER.
Inputs Database file path

Cause of failure

Dlls are loaded,File absent


EnableDisableHandlers

This function will enable or disable different handlers depending upon the argument passed to it.
Inputs Enum eHandlerType { all = 0, timers = all + 1, key = timers + 1, message = key + 1, error = message + 1 }

Cause of failure

Invalid Enum type.
