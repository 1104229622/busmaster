Hot Keys


Hot keys are available for most frequently used operations. Following are the hot keys implemented.

1. CTRL + ALT + C  ---> Clear Message Window

2. CTRL + ALT + I   ---> Toggle between Activate/Deactivate message interpretation mode.

3. CTRL + ALT + O --->  Toggle between Append/Overwrite display mode 

4. CTRL + ALT + R  ---> Reset Message Window column header positions to default.

5. CTRL + ALT + H  ---> Toggle between Hex/Dec display format.

6. CTRL + ALT + S  ---> Toggle betweenStart/Stop sending messages cyclically.

7. CTRL + ALT + L  ---> Toggle betweenEnable/Disable logging of messages.
