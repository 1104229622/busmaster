BUSMASTER is an Open Source Software tool to simulate, analyze and test data bus systems such as CAN.

 

Download, run, study, redistribute and modify. Join the BUSMASTER community!

Coding Contest 2014 Winners

Rank #1: Ethernet and customer protocol support
With Ashwin's contribution BUSMASTER is able to monitor and analyze Ethernet communication, as well as to construct new messages. Using custom protocol support access higher protocol layers is possible for sending and receiving, such as IPv4, TCP/UDP. The Ethernet hardware is addresses through the WinPcap library, well known from tcpdump and WireShark. Link: https://github.com/RBEI-AshwinRU/busmaster

 

Rank #2: “Associate different databases per channel” and “Test Automation editor channel selection”
Andreas's contribution addressed two feature requests, of which one of them was open right from the beginning of the BUSMASTER project. With closing “issue #13 Associate different databases per channel” it's possible to have different databases associated with each channel. The second issue #711 allows to use multiple channels in the Test Automation framework, making it suitable for gateway tests.
Link: https://github.com/GT-Derka/busmaster

Coding Contest 2014


Description

The coding contest is designed to encourage and inspire students, freelancers, and programmers to begin participating in the open source development of BUSMASTER, help identify and bring in new contributors, and provide students interested in computer science and other fields with exposure to real-world software development, including coding work, documentation and open source community involvement.

 

The contest will consist of a 1-month registration phase running April 1 – 30, 2014 and a 3-month development phase running May 1 – July 31, 2014.

The registration period was extended by 1-month till May 31, 2014.
