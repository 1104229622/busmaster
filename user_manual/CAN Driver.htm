CAN Driver


BUSMASTER communicates with interfacing hardware through a kernel mode device driver called CANDriver. For USB third party driver is used.

While installing the CANDriver its service type will be set to manual mode hence the driver has to be started before using the tool.

Follow the steps below to start the Parallel Port CANDriver if BUSMASTER is installed on Window 2000 or WinXP.

  • Select Start Programs  BUSMASTER  BUSMASTER Driver manager. An application as shown below comes up.
  • Click on Start Button to start the BUSMASTER driver. You can choose to start the driver in Demand(Manual) Mode or Automatic mode. In Demand mode you have to start the driver every time system is shutdown and started. But in Automatic mode the driver is started automatically when system is started.

Note

  • No need to start USB driver manually. The driver will be started automatically during Windows start-up.
  • User must be logged on as an administrator or a member of the Administrators group in order to start/stop driver and to use BUSMASTER application. Application start will fail if the user account doesn’t have admin privilege.
