EXTERNAL TOOLS

External Tools

Currently three external tools are integrated with BUSMASTER. These tools can be used only from BUSMASTER.

1. CAPL TO C Converter: Converts CAPL (*.can) file into BUSMASTER C (*.c) file.

2. DBC TO DBF Converter: Converts DBC (*.dbc) file into BUSMASTER database (*.dbf) file.

3. DBF TO DBC Converter: Converts BUSMASTER database (*.dbf) file into DBC (*.dbc) file.
